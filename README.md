## Project Description
Rally is a social calendar app designed to help people actually go do fun things with their friends.

Every day, people discover events through TikTok and Instagram - concerts, pop-ups, run clubs, food festivals - but those ideas usually get lost as screenshots or forgotten notes. Rally turns that moment of inspiration into a real plan. It bridges the gap between online discovery and real-world experiences.

Instead of saving screenshots and never acting on them, users can upload event screenshots directly into Rally. Our app parses the image and automatically adds the event to a dedicated Social Calendar, creating a centralized place for all the things you want to do.

How it works:
1. Discover a fun event on TikTok or Instagram  
2. Take a screenshot  
3. Upload the screenshot to Rally  
4. Rally parses the image (event name, date, time, location, description)  
5. The event is added to your Social Calendar  

From there, users can view all upcoming ideas in one place and easily plan with friends.

Rally was built for the **Health & Wellness** track with a focus on improving social and mental well-being.

- Encourages real-world social interaction  
- Reduces passive screen time by turning online discovery into offline action  
- Lowers the friction of making plans with friends  
- Promotes connection, novelty, and shared experiences  

By making it easier to follow through on social plans, Rally helps users spend more time outside, connected, and engaged.

People don’t lack ideas—they lack follow-through.

Rally helps turn:
- “That looks fun” → “We actually went”
- Online inspiration → Real-life experiences

Rally your friends. Stop scrolling. Start living.

## Technologies Used
- **Frontend / Fullstack:** React, Next.js, Node.js, Tailwind CSS
- **Backend / Services:** Supabase
- **AI / LLM:** Google GenAI (`@google/genai`)
- **Deployment:** Vercel

## Additional Information
- **Usage:** Rally is designed to work smoothly on mobile. You can leverage the native Share menu to send screenshots directly to the app using our PWA features.
- **Setup:** Run `npm install` and `npm run dev` in the `Frontend` directory to start the local development server.
